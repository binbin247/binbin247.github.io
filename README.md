# Binbin Nie Academic Homepage

This repository contains the bilingual academic homepage for Binbin Nie.

- English homepage: `/`
- Chinese homepage: `/zh/`
- Framework: Jekyll, based on the AcadHomepage template
- Public profile source: curated from the Feishu profile document, with private IDs, phone numbers, certificate files, grade/ranking records, and private attachment links excluded

## Local Preview

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000`.

## Publishing

After reviewing the local site, push this repository to `binbin247/binbin247.github.io` to publish it through GitHub Pages.
